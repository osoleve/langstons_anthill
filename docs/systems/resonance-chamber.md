# Crystal Resonance Chamber

*"The crystals sing"*

A chamber that generates nutrients from crystal vibrations, bypassing the death economy.

## Stats

| Property | Value |
|----------|-------|
| **Type** | Generator |
| **Generates** | Nutrients: 0.05/tick |
| **Consumes** | Nothing |
| **Location** | (3, 0) |

## Building Cost

| Resource | Amount |
|----------|--------|
| Crystals | 10 |

## Notes

This was the first system built to escape the death economy. Before it existed, all nutrients came from corpses. Now nutrients flow from vibration alone.

The resonance chamber represents a choice: use crystals for beauty (crafting), or use them for survival (nutrients).

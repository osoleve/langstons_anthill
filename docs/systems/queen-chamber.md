# Queen's Chamber

*"A warm, dark chamber where new life begins"*

The Queen produces new ants, consuming nutrients and fungus.

## Stats

| Property | Value |
|----------|-------|
| **Type** | Spawner |
| **Location** | (0, -1) |

## Spawning Mechanics

The Queen spawns new ants when:

- Population is below threshold
- Resources are sufficient
- Cooldown has passed

## Ant Types

The Queen can spawn:

- **Workers** - General-purpose ants
- **Undertakers** - Corpse processors (spawned when needed)

## Notes

The Queen is never seen. Only her output is visible: new ants appearing in the colony.

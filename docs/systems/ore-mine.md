# Ore Mine

*"Glittering Vein"*

Veins of copper and gold thread through the stone.

## Stats

| Property | Value |
|----------|-------|
| **Type** | Generator |
| **Generates** | Ore: 0.002/tick |
| **Consumes** | Nothing |
| **Location** | (-1, -1) |

## Notes

Ore has no practical purpose. It does not feed. It does not build. It exists to be beautiful.

Ore is crafted into jewelry at the Crafting Hollow.

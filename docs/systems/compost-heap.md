# Compost Heap

*"The Heap"*

Decay as resource. Dirt breaks down into nutrients.

## Stats

| Property | Value |
|----------|-------|
| **Type** | Generator |
| **Generates** | Nutrients: 0.01/tick |
| **Consumes** | Dirt: 0.005/tick |
| **Location** | (1, 0) |

## Corpse Boosts

When corpses are added to the heap, nutrient production increases temporarily.

## Notes

The compost heap closes the loop between death and life. It is where dirt becomes food becomes ant becomes corpse becomes dirt again.
